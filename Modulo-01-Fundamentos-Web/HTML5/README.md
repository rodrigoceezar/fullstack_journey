## 📚 Teoria Resumida

### Estrutura Básica
```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>Minha Página</title>
</head>
<body>
  <h1>Olá Mundo</h1>
</body>
</html>
```

```html
Principais Conceitos

Tags de Texto: <h1> até <h6>, <p>, <strong>, <em>.

Listas: <ul>, <ol>, <li>.

Links/Imagens: <a>, <img>.

Semântica: <header>, <nav>, <section>, <footer>.

Formulários: <form>, <input>, <label>, <button>.

Tabelas: <table>, <tr>, <th>, <td>.
```
